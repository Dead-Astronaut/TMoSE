1. Set Up

The Setup Guide

A step-by-step checklist to get your local environment ready for development.

Install the Engine (Python)

Action: Download the latest version from python.org.

Crucial Step: During installation, check the box that says "Add Python to PATH." This allows your computer to run Python commands from anywhere.

Install the Cockpit (VS Code)

Action: Download and install Visual Studio Code from code.visualstudio.com.

The Bridge: Open VS Code, go to the Extensions view (the square icon on the left), and install the "Python" extension by Microsoft. This teaches VS Code how to read and run your Python files.

The Hangar (GitHub) — Optional

Action: Create a free account at github.com.

Purpose: This is where you will eventually back up your code. You don't need to master this on Day 1, but having the account ready is the first step toward professional coding.

The First Flight (Hello World)

Action: Create a new file in VS Code named hello.py.

Code: Type print("Ready for PCEP!").

Run: Click the Play button in the top-right corner. If you see that text in the terminal at the bottom, your environment is 100% ready.
