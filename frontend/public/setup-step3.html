2. Onramp

The On-Ramp

This defines the conceptual "mental model" for a PCEP candidate.

  • Interpreted vs. Compiled: Python is an interpreted language, meaning an interpreter executes your code line-by-line rather than translating the whole thing at once.
  • The Rules: Lexis (the allowed vocabulary), Syntax (the grammar/arrangement), and Semantics (the actual logic/meaning).
  • Integers (int): Whole numbers (e.g., 5, -10).
  • Floats (float): Numbers with decimals (e.g., 3.14).
  • Strings (str): Text wrapped in quotes (e.g., "Hello").
  • Booleans (bool): Logic values, either True or False.
  • Variables: Named slots in memory used to store data. They follow specific naming conventions and PEP-8 style rules.
  • Lists: Ordered collections that can be changed (mutable).
  • Tuples: Ordered collections that cannot be changed (immutable).
  • Dictionaries: Key-value pairs used to organize labeled data.
  • Operators: Tools for math (+, -, *, /), comparison (==, !=, >, <), and logic (and, or, not).
  • Control Flow: The "decision-making" parts of code. Using if/else blocks to branch paths and for/while loops to repeat tasks.
